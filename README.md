
# Netflix-Analysis

### Project Overview
This project involves the analysis and visualization of Netflix content data to uncover key trends in library growth and content distribution.

### Tools Used
* **Python (Pandas):** Used for data cleaning, transformation, and preparation.
* **Power BI:** Used for creating the interactive dashboard to visualize content insights.

### Key Insights
* **Content Ratio:** Analysis of the distribution between Movies and TV Shows.
* **Release Patterns:** Identification of yearly trends in content addition.
* **Category Trends:** Mapping popular genres across the global catalog.

### Project Files
* **netflix_analysis.ipynb**: The Jupyter Notebook containing the end-to-end data cleaning and analysis code.
* **netflix_dashboard.pbix**: The final interactive Power BI dashboard file.
